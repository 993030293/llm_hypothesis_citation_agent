# Submission Package Index

This directory is the classroom-facing evidence package for the revised
Project A system: a DeepScientist-style research agent for hypothesis
generation and citation verification.

## Canonical demo evidence

- `evidence/success_case/`: copied from `outputs/runs/20260529_193717`.
  This run contains Green and Yellow citation labels.
- `evidence/boundary_case/`: copied from `outputs/runs/20260529_193944`.
  This run contains Yellow and Red citation labels.

Each evidence directory includes the core artifacts required for inspection:

- `tool_calls.jsonl`
- `retrieved_literature.jsonl`
- `citation_verification.csv`
- `final_report.md`
- `generated_hypothesis.md`
- the remaining workflow artifacts needed to reproduce the report context.

## Documents

- `reproducibility.md`: install, run, and artifact reproduction guide.
- `grading_checklist.md`: rubric-to-evidence mapping.
- `failure_case_explanation.md`: explanation of Yellow and Red boundary cases.
- `demo_script.md`: 10-minute demo script and backup plan.
- `rehearsal_checklist.md`: checklist for the required full rehearsal.
- `teacher_live_pdf_protocol.md`: preflight and live-demo procedure for a PDF
  supplied by the teacher during presentation.
- `../docs/qq_live_integration.md`: real NapCat/OneBot QQ integration guide.
- `../docs/stress_audit.md`: batch stress test and online verification guide.
- `stress_audit/`: generated by `scripts/stress_audit.py` after a full or smoke
  audit run.
- `poster/poster_120x80.md`: editable poster source.
- `poster/poster_120x80.pdf`: generated 120 cm x 80 cm poster PDF.
